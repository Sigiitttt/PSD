# Proyek Sains Data (PSD)

Repositori ini berisi tugas mata kuliah **Proyek Sains Data (PSD)** yang mencakup proses end-to-end dalam pengolahan data hingga pembangunan model machine learning. Seluruh tahapan mengikuti standar alur kerja data science mulai dari pengumpulan data, eksplorasi, pembersihan, pemodelan, evaluasi, hingga deployment sederhana.

## 🎯 Tujuan Proyek

* Menerapkan seluruh tahapan *data science lifecycle* pada satu studi kasus.
* Mengembangkan kemampuan analisis data, pembuatan model prediktif, dan interpretasi hasil.
* Menyusun dokumentasi dan pelaporan yang rapi sesuai standar akademik.
* Mempresentasikan hasil akhir proyek dalam bentuk aplikasi, dashboard, atau laporan komprehensif.

## 📂 Struktur Umum Proyek

Proyek ini umumnya terdiri dari beberapa komponen penting:

### 1. **Dataset**

Berisi data mentah yang digunakan dalam analisis. Format dapat berupa CSV, Excel, JSON, atau lainnya.

### 2. **Notebook Analisis** (`.ipynb`)

Tahapan-tahapan utama:

* Import data
* Data cleansing
* EDA (Exploratory Data Analysis)
* Feature engineering
* Pemilihan dan pelatihan model
* Evaluasi model

### 3. **Script Pemodelan** (`.py`)

Berisi fungsi penting seperti preprocessing, training, dan inference model.

### 4. **Model** (`.pkl` atau `.h5`)

Model machine learning yang sudah dilatih dan siap digunakan.

### 5. **Aplikasi**

Jika tugas mengharuskan deployment, dapat berupa:

* Streamlit
* Dashboard visualisasi

### 6. **Dokumentasi**

Berisi penjelasan lengkap mengenai hasil proyek.

## 🧪 Metodologi Sains Data

Proyek PSD mengikuti alur:

1. **Problem Definition** – Mendefinisikan tujuan proyek.
2. **Data Collection** – Pengambilan data internal/eksternal.
3. **Data Cleaning** – Handling missing values, outliers, duplikasi, dan normalisasi.
4. **EDA (Exploratory Data Analysis)** – Analisis pola dan karakteristik data.
5. **Feature Engineering** – Transformasi dan pembuatan fitur baru.
6. **Modeling** – Mencoba beberapa algoritma machine learning.
7. **Evaluation** – Menggunakan metrik seperti accuracy, F1-score, RMSE, dll.
8. **Deployment / Presentation** – Menyajikan hasil proyek dalam bentuk aplikasi atau laporan.
