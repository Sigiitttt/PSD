Business Understanding#

1. Latar Belakang Masalah#

Perkembangan teknologi di bidang biomedis dan analitik data memungkinkan pemanfaatan sinyal fisiologis manusia sebagai sumber informasi untuk berbagai kebutuhan, seperti identifikasi individu, sistem keamanan, dan pemantauan kesehatan. Salah satu sinyal fisiologis yang banyak diteliti adalah surface Electromyography (sEMG), yaitu sinyal listrik yang dihasilkan oleh aktivitas otot dan direkam melalui permukaan kulit.

Setiap individu memiliki karakteristik otot dan pola aktivitas listrik yang berbeda, sehingga sinyal sEMG berpotensi digunakan sebagai ciri biometrik. Namun, data sEMG bersifat kompleks, berdimensi tinggi, dan dipengaruhi oleh berbagai faktor, sehingga diperlukan pendekatan Data Science dan Machine Learning untuk mengekstraksi pola yang bermakna dari data tersebut.

Dataset SemgHandSubjectCh2 menyediakan data sEMG berbasis spektrum daya dari beberapa subjek yang melakukan aktivitas genggaman tangan. Dataset ini dapat dimanfaatkan untuk membangun model klasifikasi yang mampu mengidentifikasi subjek berdasarkan karakteristik sinyal sEMG.

2. Permasalahan Bisnis (Business Problem)#

Dalam konteks dunia nyata, terdapat kebutuhan akan sistem yang mampu:

  • Mengidentifikasi individu secara non-invasif

  • Memanfaatkan sinyal fisiologis alami tanpa memerlukan input eksplisit dari pengguna

  • Bekerja dengan data sinyal biologis yang kompleks

Namun, tanpa analisis data yang tepat, sinyal sEMG sulit digunakan secara langsung karena:

  • Memiliki variabilitas tinggi

  • Bersifat non-linear

  • Mengandung noise

Oleh karena itu, diperlukan sistem berbasis Data Science yang mampu mengklasifikasikan identitas subjek secara akurat berdasarkan sinyal sEMG.

3. Tujuan Proyek (Business Objectives)#

Tujuan utama dari proyek ini adalah:

Membangun dan mengevaluasi model klasifikasi berbasis Data Science yang mampu mengidentifikasi subjek berdasarkan sinyal sEMG Channel 2 dengan tingkat akurasi yang baik.

Tujuan khusus meliputi:

  1. Memahami karakteristik data sEMG berbasis spektrum daya

  2. Melakukan pra-pemrosesan data agar siap digunakan untuk pemodelan

  3. Membangun model klasifikasi multi-kelas (5 kelas subjek)

  4. Mengevaluasi performa model menggunakan metrik evaluasi yang sesuai

  5. Menentukan model terbaik berdasarkan hasil evaluasi

4. Manfaat Bisnis (Business Value)#

Hasil dari proyek ini diharapkan memberikan manfaat sebagai berikut:

Manfaat Praktis

  • Menjadi dasar pengembangan sistem identifikasi biometrik berbasis sEMG

  • Mendukung penelitian di bidang biomedis, wearable devices, dan human–computer interaction

  • Memberikan gambaran penerapan Data Science pada data sinyal fisiologis

Manfaat Akademis

  • Menjadi studi kasus penerapan metode klasifikasi machine learning

  • Menambah pemahaman tentang pengolahan data time-series dan signal-based data

  • Melatih kemampuan analisis data kompleks dalam konteks dunia nyata

5. Ruang Lingkup Proyek (Scope)#

Agar proyek tetap terfokus, ruang lingkup proyek dibatasi sebagai berikut:

  • Dataset yang digunakan: SemgHandSubjectCh2

  • Data yang dianalisis: hasil ekstraksi spektrum daya (bukan sinyal mentah)

  • Channel yang digunakan: Channel 2

  • Jumlah kelas: 5 subjek

  • Metode yang digunakan: klasifikasi berbasis Machine Learning / Deep Learning

  • Proyek ini tidak mencakup implementasi perangkat keras atau sistem real-time

6. Kriteria Keberhasilan (Success Criteria)#

Proyek dikatakan berhasil apabila:

  1. Model mampu melakukan klasifikasi subjek dengan performa yang baik

  2. Nilai akurasi dan metrik evaluasi lainnya menunjukkan hasil yang konsisten

  3. Model dapat menggeneralisasi data uji dengan baik (tidak overfitting)

  4. Hasil analisis dapat dijelaskan secara logis dan ilmiah

7. Ringkasan Business Understanding#

Secara keseluruhan, proyek ini bertujuan untuk memanfaatkan data sEMG sebagai sumber informasi biometrik melalui pendekatan Data Science. Dengan membangun model klasifikasi yang tepat, diharapkan sistem dapat mengenali identitas subjek berdasarkan pola aktivitas otot tangan secara efektif.

Data Understanding#

1. Deskripsi Dataset#

Dataset yang digunakan dalam proyek ini adalah SemgHandSubjectCh2, yang merupakan bagian dari dataset SEMG for Basic Hand Movements yang tersedia pada UCI Machine Learning Repository. Dataset ini berisi data surface Electromyography (sEMG) yang direkam dari aktivitas otot tangan manusia menggunakan sistem EMG dua kanal, di mana pada proyek ini hanya digunakan Channel 2.

Data yang digunakan bukan berupa sinyal mentah, melainkan hasil ekstraksi spektrum daya (power spectrum) dari sinyal sEMG, sehingga data berada pada domain frekuensi.

sumber dataset https://www.timeseriesclassification.com/description.php?Dataset=SemgHandSubjectCh2

Selain itu, urutan data tidak memiliki makna tertentu, sehingga setiap sampel dianggap independen satu sama lain.

5. Tipe dan Karakteristik Data#

Berdasarkan bentuk dan sifatnya, data memiliki karakteristik sebagai berikut:

  • Numerik kontinu

  • Berdimensi tinggi (1500 fitur per sampel)

  • Berbasis sinyal frekuensi

  • Memiliki potensi korelasi antar fitur

Karakteristik ini mengindikasikan perlunya:

  • Normalisasi atau standarisasi data

  • Teknik reduksi dimensi (opsional)

  • Model yang mampu menangani data berdimensi tinggi

6. Kualitas Data#

Berdasarkan deskripsi dataset:

  • Data tidak mengandung nilai kosong (missing values)

  • Data telah melalui proses ekstraksi spektrum daya

  • Noise relatif lebih rendah dibandingkan sinyal mentah

Namun demikian, tetap diperlukan:

  • Pemeriksaan nilai ekstrem (outliers)

  • Analisis statistik dasar untuk memahami rentang nilai data

  • Visualisasi data untuk memahami pola awal

7. Eksplorasi Awal Data (Initial Data Exploration)#

Pada tahap awal eksplorasi data, analisis yang dapat dilakukan meliputi:

  1. Analisis statistik deskriptif (mean, standar deviasi, minimum, maksimum)

  2. Visualisasi sinyal spektrum daya untuk beberapa sampel dari tiap kelas

  3. Perbandingan pola spektrum antar subjek

  4. Analisis variansi fitur antar kelas

Tujuan eksplorasi ini adalah untuk:

  • Mengidentifikasi perbedaan karakteristik antar kelas

  • Menilai kompleksitas data

  • Menentukan pendekatan pemodelan yang tepat

8. Tantangan Data (Data Challenges)#

Beberapa tantangan utama dalam dataset ini antara lain:

  • Dimensi data yang tinggi dibandingkan jumlah sampel

  • Potensi redundansi antar fitur spektrum

  • Variabilitas sinyal antar individu

  • Risiko overfitting jika model terlalu kompleks

Tantangan-tantangan ini menjadi pertimbangan penting dalam pemilihan metode dan strategi pemodelan.

9. Ringkasan Data Understanding#

Dataset SemgHandSubjectCh2 merupakan dataset sinyal sEMG berbasis spektrum daya dengan karakteristik numerik berdimensi tinggi dan target multi-kelas. Dataset ini cocok untuk diterapkan metode klasifikasi berbasis Data Science, dengan tetap memperhatikan tantangan seperti korelasi fitur dan risiko overfitting. Pemahaman yang baik terhadap karakteristik data menjadi dasar penting sebelum memasuki tahap Data Preparation dan Modeling.

def load_ts_file(file_path):
    encodings = ['utf-8', 'latin-1']
    data = []

    for enc in encodings:
        try:
            with open(file_path, 'r', encoding=enc) as file:
                is_data = False
                for line in file:
                    line = line.strip()
                    if not line or line.startswith(('#', '%')): continue

                    if line.lower().startswith("@data"):
                        is_data = True
                        continue

                    if not is_data: continue
                    if ':' in line:
                        parts = line.split(':')
                        features_str = parts[0]
                        label_str = parts[-1]

                        values = features_str.split(',')
                        values.append(label_str)
                    else:
                        values = line.split(',')

                    data.append(values)
            break
        except UnicodeDecodeError:
            continue

    if not data:
        raise ValueError(f"Gagal membaca file {file_path}.")

    df = pd.DataFrame(data)

    X = df.iloc[:, :-1].astype(float)
    y = df.iloc[:, -1]

    return X, y

def load_ts_file(file_path):
    encodings = ['utf-8', 'latin-1']
    data = []

    for enc in encodings:
        try:
            with open(file_path, 'r', encoding=enc) as file:
                is_data = False
                for line in file:
                    line = line.strip()
                    if not line or line.startswith(('#', '%')): continue

                    if line.lower().startswith("@data"):
                        is_data = True
                        continue

                    if not is_data: continue

                    if ':' in line:
                        parts = line.split(':')
                        features_str = parts[0]
                        label_str = parts[-1]

                        values = features_str.split(',')
                        values.append(label_str)
                    else:
                        # Jika tidak ada titik dua, asumsi dipisah koma semua
                        values = line.split(',')

                    data.append(values)
            break
        except UnicodeDecodeError:
            continue

    if not data:
        raise ValueError(f"Gagal membaca file {file_path}.")

    # Konversi ke DataFrame
    df = pd.DataFrame(data)

    X = df.iloc[:, :-1].astype(float)
    y = df.iloc[:, -1]

    return X, y
