Preprocessing Data IRIS: Penghapusan Outlier dengan Tiga Metode#

Pendahuluan#

Setelah tahap identifikasi outlier dilakukan, langkah selanjutnya adalah membersihkan data dari nilai-nilai anomali yang dapat mengganggu analisis atau proses pemodelan.
Pada bagian ini, saya menerapkan tiga metode berbeda untuk menghapus outlier, yaitu:

  1. LOF (Local Outlier Factor)

  2. KNN (K-Nearest Neighbors)

  3. ABOD (Angle-Based Outlier Detection)

Tujuannya adalah membandingkan hasil dari masing-masing metode agar dapat dipahami perbedaan jumlah data yang dianggap normal maupun outlier.

Metodologi#

1. Local Outlier Factor (LOF)#

  • LOF mendeteksi anomali berdasarkan kepadatan lokal sebuah titik dibandingkan tetangganya.

  • Titik dengan kepadatan jauh lebih rendah dianggap outlier.

2. K-Nearest Neighbors (KNN)#

  • KNN menilai suatu titik sebagai outlier bila jaraknya ke tetangga terdekat relatif lebih jauh dibandingkan data lain.

  • Pendekatan ini efektif jika distribusi data berbentuk kluster.

3. Angle-Based Outlier Detection (ABOD)#

  • ABOD menganalisis sudut antar vektor yang dibentuk sebuah titik dengan titik lain.

  • Titik yang sudutnya menyebar tidak normal (variansi besar) cenderung dikategorikan sebagai outlier.

  • Cocok untuk dataset berdimensi rendah–menengah seperti Iris.

Hasil Analisis Outlier Removal#

Setelah setiap metode dijalankan, dataset dibagi menjadi dua kategori:

  • Normal (Anomaly = 0) → Data yang dianggap wajar.

  • Outlier (Anomaly = 1) → Data yang menyimpang.

  • Titik hijau → data normal.

  • Titik oranye → outlier.

Insight dan Kesimpulan#

  1. Ketiga metode konsisten mendeteksi 8 outlier dari total 150 data.

  2. Data bersih yang tersisa adalah 142 baris, sehingga dataset tetap representatif untuk analisis lebih lanjut.

  3. Konsistensi ini menunjukkan bahwa outlier pada dataset Iris cukup jelas sehingga dapat terdeteksi dengan berbagai algoritma.

  4. Dengan data yang sudah bersih, proses pemodelan machine learning akan lebih akurat karena tidak terganggu oleh data ekstrem.

✍️ Sebagai penulis, saya menilai bahwa langkah outlier removal ini penting untuk memastikan kualitas data sebelum masuk ke tahap analisis atau pemodelan.
Pada kasus Iris, ketiga metode menunjukkan hasil yang sama, sehingga kita bisa lebih percaya diri bahwa 8 data yang terdeteksi memang benar merupakan outlier.#

Cuplikan Code#

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from pycaret.anomaly import *

# 1. Load Data
data = pd.read_csv("data_iris.csv")

print("=== Informasi Dataset ===")
print(data.info())
print("\n=== 5 Data Teratas ===")
print(data.head())

# Nomor 2: Hapus Outlier dengan 3 Metode
def remove_outliers(method="lof"):
    model = create_model(method)
    results = assign_model(model)

    # Gabungkan kembali
    data_with_outlier = data.copy()
    data_with_outlier["Anomaly"] = results["Anomaly"]

    # Hitung jumlah
    n_outliers = (data_with_outlier['Anomaly'] == 1).sum()
    n_normal = (data_with_outlier['Anomaly'] == 0).sum()

    # Hapus outlier
    clean_data = data_with_outlier[data_with_outlier["Anomaly"] == 0].drop(columns=["Anomaly"])

    print(f"\n=== Metode: {method.upper()} ===")
    print("Jumlah Outlier   :", n_outliers)
    print("Jumlah Normal    :", n_normal)
    print("Data bersih sisa :", len(clean_data))

    # Visualisasi hasil (diperbaiki supaya jumlah sesuai print)
    plt.figure(figsize=(8,6))

    # Plot data normal (hijau)
    sns.scatterplot(
        x=data_with_outlier.loc[data_with_outlier["Anomaly"]==0, "sepal length"],
        y=data_with_outlier.loc[data_with_outlier["Anomaly"]==0, "sepal width"],
        color="green", alpha=0.6, label=f"Normal ({n_normal})"
    )

    # Plot data outlier (orange, marker 'X')
    sns.scatterplot(
        x=data_with_outlier.loc[data_with_outlier["Anomaly"]==1, "sepal length"],
        y=data_with_outlier.loc[data_with_outlier["Anomaly"]==1, "sepal width"],
        color="orange", marker="X", s=100, alpha=0.9, label=f"Outlier ({n_outliers})"
    )

    plt.title(f"Outlier Detection - {method.upper()}")
    plt.xlabel("Sepal Length")
    plt.ylabel("Sepal Width")
    plt.legend(title="Kategori")
    plt.grid(alpha=0.3)
    plt.savefig(f"outlier_{method}.png")
    plt.show()

    return clean_data


# Jalankan 3 metode
clean_lof = remove_outliers("lof")
clean_knn = remove_outliers("knn")
clean_abod = remove_outliers("abod")

summary = pd.DataFrame({
    "Metode": ["LOF", "KNN", "ABOD"],
    "Jumlah Data Awal": [len(data)]*3,
    "Jumlah Normal": [len(clean_lof), len(clean_knn), len(clean_abod)],
    "Jumlah Outlier": [
        len(data)-len(clean_lof),
        len(data)-len(clean_knn),
        len(data)-len(clean_abod)
    ]
})

print("\n=== Ringkasan Perbandingan Outlier Removal ===")
print(summary)

# Simpan hasil ke file
summary.to_csv("outlier_summary.csv", index=False)
clean_lof.to_csv("iris_clean_lof.csv", index=False)
clean_knn.to_csv("iris_clean_knn.csv", index=False)
clean_abod.to_csv("iris_clean_abod.csv", index=False)
